Enunciado

Clara, uma analista de dados, está trabalhando em um projeto de pesquisa onde precisa encontrar a mediana de conjuntos de dados. A mediana é crucial para determinar a tendência central dos dados sem ser afetada por valores extremos. Você recebeu uma lista não vazia de números inteiros. Separe a metade superior dos números da metade inferior e encontre a mediana. Em uma lista com um número ímpar de elementos, a mediana é o número encontrado no meio da lista. Se a lista contiver um número par de elementos, a mediana será a média dos dois números do meio.

Entrada: Uma lista não vazia de números inteiros.

Saída: A mediana como um número inteiro ou float.
