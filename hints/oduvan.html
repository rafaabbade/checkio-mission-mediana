Como ordenar uma lista?

O método sorted() pode ser usado para ordenar uma lista em ordem crescente. Ele retorna uma nova lista ordenada. 

lista = [3, 1, 4, 1, 5]
lista_ordenada = sorted(lista)
print(lista_ordenada)  # Saída: [1, 1, 3, 4, 5]
